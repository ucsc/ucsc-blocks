
# UCSC Blocks

Custom blocks for UCSC WordPress websites.

## Documentation

Usage guides for the blocks in this plugin:

- **UCSC Events** — display upcoming campus events by organizer. See [docs/ucsc-events/setup.md](docs/ucsc-events/setup.md).
- **Calendar Feed** — display upcoming events from an iCalendar (`.ics`) feed. See [docs/calendar-feed/setup.md](docs/calendar-feed/setup.md).
